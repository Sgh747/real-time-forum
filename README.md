## 🗺️ О проекте

**Real-Time Forum** — это живое городское пространство в цифровом мире. Здесь пользователи встречаются, обсуждают идеи и общаются в режиме реального времени. Проект вдохновлён метафорой города: чтобы войти — нужно пройти через ворота, а внутри вас ждут площади для дискуссий и личные переговорные комнаты.

Форум построен на чистом **Go** без сторонних фреймворков, с использованием **WebSocket** для мгновенного обмена сообщениями и параллакс-эффектами для атмосферного пользовательского опыта.


## ✨ Возможности

### 🏠 Навигация
| Раздел | Описание |
|--------|----------|
| **Главная** | Лендинг с параллакс-эффектом при прокрутке |
| **Посты** | Лента публикаций с категориями, лайками и дизлайками |
| **Чат** | Общий и приватные чаты в реальном времени |
| **Вход / Регистрация** | Авторизация через городские ворота |

### 📝 Посты
- ✅ Создание, редактирование и удаление постов
- 🏷️ Категории для удобной навигации
- 👍 Система лайков и дизлайков
- 🔒 Создание постов доступно только авторизованным пользователям

### 💬 Чат
- 🌐 **Общий чат** — доступен для всех зарегистрированных пользователей
- 🔒 **Приватные чаты** — создайте личное пространство для избранных
- 📨 **Приглашения** — пригласите участника правым кликом по нику
- 🔍 **Поиск** — найдите любого участника по никнейму
- 🚪 Создание собственных чат-комнат

## 🚀 Быстрый старт


### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/your-username/real-time-forum.git

# 2. Перейдите в директорию проекта
cd real-time-forum

# 3. Запустите сервер
go run ./cmd/server
```

После запуска откройте браузер и перейдите по адресу:

```
 http://localhost:8080/forum#home
```

## 💬 Как пользоваться чатом

### Приватный чат — пошагово:

```
1. Нажмите кнопку  [+ Создать приватный чат]

2. В общем чате найдите нужного пользователя

3. Кликните ПРАВОЙ кнопкой мыши на никнейм участника

4. Выберите «Пригласить в приватный чат»

5. Ожидайте — участник получит приглашение

6. После принятия приглашения пользователь окажется
   в вашем приватном чате 🎉
```

> 💡 **Совет:** Используйте поиск по никнейму, чтобы быстро найти нужного участника в большом чате.
